Introdução

A princípio, para o desenvolvimento deste site,
são necessários três itens fundamentais, que
servirão como base para sua
estrutura e funcionamento.
  • HTML
  • CSS
  • JavaScript

HTML

HTML (HyperText Markup Language): é a
linguagem de marcação responsável por estruturar
o conteúdo de uma página na web. Ele define os
elementos básicos de um site, como títulos,
parágrafos, imagens, links e seções. Neste
projeto, o HTML será utilizado para criar a
estrutura principal do site, organizando todas
as informações e componentes
que serão exibidos ao usuário.

CSS

CSS (Cascading Style Sheets): é a
linguagem responsável por definir o estilo
e a aparência de uma página web. Ele permite
controlar cores, fontes, espaçamentos,
tamanhos, layouts e o posicionamento dos
elementos. Neste projeto, o CSS será utilizado
para melhorar a apresentação visual do site,
tornando-o mais organizado, agradável e
responsivo para diferentes tamanhos de tela.

JavaScript

JavaScript: é uma linguagem de programação
utilizada para adicionar interatividade e dinamismo às
páginas web. Ele permite criar comportamentos como
cliques, animações, validações de formulários e
atualizações de conteúdo sem precisar recarregar a página.
Neste projeto, o JavaScript será utilizado para tornar
o site mais interativo, permitindo ações do
usuário e melhorando a experiência de navegação.

TAGS

Estrutura básica do documento

  • <html>: Elemento raiz da página HTML.
  • <head>: Contém metadados, links e scripts.
  • <body>: Conteúdo visível da página.
  • <title>: Título da aba do navegador.
  • <meta>: Metadados da página.
  • <link>: Liga arquivos externos como CSS.
  • <style>: CSS interno.
  • <script>: JavaScript.
  • <base>: URL base para links relativos.

Estrutura semântica

  • <header>: Cabeçalho da página ou seção.
  • <nav>: Área de navegação.
  • <main>: Conteúdo principal.
  • <section>: Seção de conteúdo.
  • <article>: Conteúdo independente.
  • <aside>: Conteúdo lateral.
  • <footer>: Rodapé da página.
  • <figure>: Conteúdo ilustrativo.
  • <figcaption>: Legenda da figura.
  • <details>: Conteúdo expansível.
  • <summary>: Título do elemento details.
  • <dialog>: Caixa de diálogo.

Texto e formatação

  • <h1> até <h6>: Títulos.
  • <p>: Parágrafo.
  • <br>: Quebra de linha.
  • <hr>: Linha horizontal.
  • <span>: Elemento inline.
  • <div>: Bloco genérico.
  • <pre>: Texto pré-formatado.
  • <blockquote>: Citação em bloco.
  • <q>: Citação curta.
  • <abbr>: Abreviação.
  • <address>: Endereço.
  • <cite>: Referência.
  • <b>: Negrito visual.
  • <strong>: Importância forte.
  • <i>: Itálico.
  • <em>: Ênfase.
  • <u>: Sublinhado.
  • <mark>: Destacado.
  • <small>: Texto pequeno.
  • <sub>: Subscrito.
  • <sup>: Sobrescrito.
  • <del>: Texto removido.
  • <ins>: Texto inserido.
  • <s>: Texto riscado.
  • <code>: Código.
  • <kbd>: Entrada do teclado.
  • <samp>: Saída de programa.
  • <var>: Variável.
  • <time>: Data e hora.

Mídia e links

  • <a>: Link.
  • <area>: Área clicável.
  • <map>: Mapa de imagem.
  • <img>: Imagem.
  • <picture>: Imagem responsiva.
  • <source>: Fonte de mídia.
  • <audio>: Áudio.
  • <video>: Vídeo.
  • <track>: Legendas.
  • <iframe>: Página incorporada.
  • <canvas>: Desenho via JS.
  • <svg>: Vetores gráficos.

Listas e tabelas

  • <ul>: Lista não ordenada.
  • <ol>: Lista ordenada.
  • <li>: Item de lista.
  • <dl>: Lista de definição.
  • <dt>: Termo.
  • <dd>: Descrição.
  • <table>: Tabela.
  • <caption>: Título da tabela.
  • <thead>: Cabeçalho da tabela.
  • <tbody>: Corpo da tabela.
  • <tfoot>: Rodapé da tabela.
  • <tr>: Linha da tabela.
  • <th>: Cabeçalho da célula.
  • <td>: Célula.

Formulários

  • <form>: Formulário.
  • <input>: Campo de entrada.
  • <textarea>: Texto longo.
  • <button>: Botão.
  • <select>: Lista suspensa.
  • <option>: Opção.
  • <optgroup>: Grupo de opções.
  • <label>: Rótulo.
  • <fieldset>: Grupo de campos.
  • <legend>: Título do grupo.
  • <datalist>: Sugestões.
  • <output>: Resultado.
  • <progress>: Progresso.
  • <meter>: Medidor.

Avançado

  • <noscript>: Sem JavaScript.
  • <template>: Modelo reutilizável.
  • <slot>: Slot Web Components.
